Drow Word Processor

As a team project in 3rd year, Two teammates (Lee Mc Donald and Graham Wolfe) and I developed a word processor, Drow.

Back View on GitHub Download the code Download the User Manual


Drow is a word processor written in Java and Java Swing.
Featuring its own file type (.drow) and many common features, Drow also has a sepcial developer mode for editing code files.

Drow is very much a student project and is not recommended to be used for actual word processing.
It is very buggy and I won't claim responsibility for any damage it might do to your system or sanity.
