About Me

I'm Simon O'Neill, a current undergraduate at DIT. I like to code, from small scripts to massive projects. The more complicated the project, the more interested I am. I'm experienced in C/C++, C# and Java with some minor experience in MatLab, Lua, Bash, HTML and PHP. Mostly I write PC applications but I've done some mobile work on Android and Windows Phone. Below I've listed some of my projects that you might find interesting. If you'd like to contact me I'm on GitHub, Linkedin and Twitter, or if you'd prefer you can always send me an e-mail.
