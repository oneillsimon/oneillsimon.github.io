G.E.M.

NOTE GEM is still a work in progress, a complete API is on the way. The Good Enough Maths library is a collection of mathematical functions useful in the development of games.

  • Download
  • GitHub

GEM contains 3 mathematical structures: vectors, matrices and quaternions. GEM is designed to be small and versatile. Making use of C++ templates, GEM is flexible, portable and sure to meet your needs.

Vectors

GEM supports vectors of any length, and any type. A GEM vector is declared simply as

gem::vector<T> v;

where T is the type of data in the vector. Vectors are designed to be used with primitive types, that is int, float, and double. GEM vectors can be of any length and are constructed using a std::intializer_list. So a both the following constructors are valid and produce the same object.

gem::vector<int> v = gem::vector<int>({ 1, 2, 3, 4, 5 });

gem::vector<int> v = { 1, 2, 3, 4, 5 };

For convnience, there are several common types of vectors defined in GEM.

gem::vector2<T> // A vector of length 2, of type T

gem::vector3<T> // A vector of length 3, of type T

gem::vector4<T> // A vector of length 4, of type T

For even further convenience, vectors of specifc lengths and types are also defined.

gem::vector2i // vector2 of integers

gem::vector2f // vector2 of floats

gem::vector2d // vector2 of doubles

gem::vector3i // vector3 of integers

gem::vector3f // vector3 of floats

gem::vector3d // vector3 of doubles

gem::vector4i // vector4 of integers

gem::vector4f // vector4 of floats

gem::vector4d // vector4 of doubles

Matrices

Matrices in GEM are row major and similar to vectors can be defined as any size. An example matrix constructor is shown.

gem::matrix<int> m = gem::matrix<int>({ { 1, 2, 3 }, { 4, 5, 6 } });

This will create a matrix of 2 rows and 3 columns. Another way to construct the same matrix would be to call.

gem::matrix<int> m = { { 1, 2, 3 }, { 4, 5, 6 } };

Also like the vector class there are several predefined matrix types for commonly used matrices.

matrix4<T> // A 4x4 matrix of type T

matrix3<T> // A 3x3 matrix of type of T

matrix3i // 3x3 matrix on ints

matrix3f // 3x3 matrix on floats

matrix3d // 3x3 matrix on doubles

matrix4i // 4x4 matrix on ints

matrix4f // 4x4 matrix on floats

matrix4d // 4x4 matrix on doubles

Quaternions

Quaternions are used for storing rotation in 3D space. A quaternion can either be constructed using 4 elements.

gem::quaternion q = quaternion(0, 0, 0, 1);

Or a quaternion can be constructed using a vector3 and a scalar value.

gem::quaternion q = quaternion(vector3f(0, 1, 0), 1);
